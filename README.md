# Automatic-File-Sorter-in-File-Explorer
## Overview

The Automatic File Sorter is a Python-based automation tool designed to organize files within a specified directory by categorizing them based on file extensions.

This solution improves digital workspace efficiency by reducing manual file management and ensuring structured directory organization.

The project demonstrates practical application of Python scripting for automation, file system manipulation, and workflow optimization.

## Key Features

Automated file categorization by extension

Dynamic folder creation for new file types

Safe file movement using built-in Python libraries

Lightweight and dependency-free implementation

Easily customizable sorting rules

## How It Works

The script scans the specified target directory.

Each file is evaluated based on its extension.

If a corresponding category folder does not exist, it is created automatically.

The file is moved into its designated folder.
